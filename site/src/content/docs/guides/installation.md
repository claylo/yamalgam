---
title: Installation
description: How to install yamalgam
---

## From crates.io

```bash
cargo install yamalgam
```

## From source

```bash
git clone https://github.com/claylo/yamalgam.git
cd yamalgam
cargo install --path crates/yamalgam
```

## Pre-built binaries

Download the latest release for your platform from the
[releases page](https://github.com/claylo/yamalgam/releases).

## Verify installation

```bash
yamalgam --version
```
